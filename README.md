# Globalify

This is the main project of the Globalify Ecosystem. It is a web app that help entrepreneurs and investors find each other easeier.

## Pre-requisites

-   Node.js 14+
-   Python 3.10+
-   Docker

## Installation

1. Clone the repository
2. Run `npm install` to install the dependencies.
3. Run `pip install poetry` to install poetry.
4. Run `poetry install` to install the python dependencies.
