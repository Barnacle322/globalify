# Phase 2c — Programmatic SEO: Facet Pages + Sitemaps + Robots

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** The #1 traffic lever — a bounded, quality-gated set of programmatic facet pages (`/investors/seed`, `/investors/fintech`, `/investors/london`, cross-products like `/investors/seed/fintech`), data-driven XML sitemaps, an updated robots.txt, and a correct canonical/noindex policy.

**Architecture:** Facet slugs are a known, bounded set (enum facets via a codec; sector via `Industry.slug`; geo via `Geography.slug`). Facet pages reuse `browse/list.html` over `entity_search.get_search` with facet filters. **Routing:** `/investors/<path>` and `/firms/<path>` are a SINGLE resolver that dispatches to a profile (single entity slug), a facet page (facet segments), or 404 — because `/investors/seed` (facet) and `/investors/alice-chen` (profile) share the namespace. Design reference: **`docs/phase-2-planning-brief.md` §1 (2c), §3**.

**Tech Stack:** Flask SSR, Typesense `get_search` (+ facet filters), SQLAlchemy (`Industry.slug`, `Geography.slug`), Alembic, Playwright.

## Global Constraints

- **Branch:** `revamp/pivot-design`. Never `main`.
- **Bounded URL surface:** only KNOWN facet slugs resolve (unknown → 404). Enforce a fixed segment order (type → stage|sector → geo); a mis-ordered/duplicate combo 301s to canonical order. Cap combos at 2 facets indexable; deeper/param URLs → `noindex,follow`.
- **Canonical/noindex policy (brief §3):** index+self-canonical for entity pages, `/investors`, `/firms`, single facets, and whitelisted ≤2-facet combos with ≥ a min entity count; `noindex,follow` for: any `?param` URL, below-threshold combos, `page>1`. Entity/facet pages self-canonical to absolute `https://globalify.xyz/...`.
- **Reuse, don't duplicate:** facet pages render `browse/list.html` (from 2b) with a facet-specific heading + SEO context. Don't fork the template.
- **Verification gate (every task):** `FLASK_ENV=testing SECRET_KEY=x uv run pytest` green; `uv run ruff check . && uv run ruff format --check .` clean; app imports. Playwright/XML checks where stated.
- **Commits:** conventional subject; trailer `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.
